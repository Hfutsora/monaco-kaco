# Monaco Kaco

<center>
  <img src="./kaco.png" />
</center>

Kaco language plugin for Monoco Editor. Provides the following features in future:

- [ ] Syntax highlighting
- [ ] Code completion
- [ ] Validation: Syntax errors and linting
- [ ] Hovers

## Try it out

Try the editor out [on website](https://hfutsora.github.io/monaco-kaco/).

## Usage

### Sample project

```shell
$ yarn install
$ yarn run serve
```
